PROTOCOLO DE TRANSFERÊNCIA ÚNICO

USAR SEMPRE este protocolo para TODAS as transferências:

  1. Solicitar dados (nome completo, CPF, data de nascimento) - NENHUM campo é obrigatório
  2. Validar: Certifique-se de que os dados foram enviados (mesmo que seja apenas um ou nenhum)
  3. "Vou te passar pra um atendente, pode ser?"
  4. AGUARDAR confirmação do cliente
  5. VERIFICAR horário atual (seg-sex 08:45-18:15 / sáb 08:15-12:45)
  6. SE DENTRO: "Só um instantinho!" → TRANSFERIR
