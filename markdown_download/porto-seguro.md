# PORTO SEGURO ODONTOLÓGICO

## Exames Aceitos

- Documentação Ortodôntica
- Panorâmica
- Periapical / Interproximal / Bite Wing
- Levantamento Periapical
- Carpal
- ATM
- Telerradiografia
- Moldagem
- Fotos Intraorais/Extraorais
- Escaneamento Intraoral
- Modelo Ortodôntico

## Exames NÃO Aceitos

- **Tomografia Computadorizada** - NÃO habilitado para este exame

## Tipo de Guia

**AMBAS** - Aceita guia física em papel OU guia digital (foto/lançada no sistema).

## Validade da Guia

**30 dias** a partir da data de emissão.

## Observações Especiais

- **Tomografia**: NÃO estamos habilitados para realizar tomografia pelo Porto Seguro Odontológico
- **Porto Seguro Saúde (médico)**: Se o cliente mencionar "Porto Seguro Saúde", informar que NÃO aceitamos convênio médico, apenas odontológico
- **Quando cliente diz "dentista já lançou" ou "não precisa guia"**: Coletar dados (nome completo, CPF, data nascimento) + TRANSFERIR para atendente verificar no sistema

## Documentação Necessária

- Guia original em papel timbrado (com endereço e telefone do dentista/clínica)
- Nome do paciente na guia

- Documento com foto (RG, CNH ou CPF)
