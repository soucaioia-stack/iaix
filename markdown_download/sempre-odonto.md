# SEMPRE ODONTO

Inclui: **CAEDU**

## Exames Aceitos

**TODOS os exames** são aceitos:
- Documentação Ortodôntica
- Panorâmica
- Tomografia Computadorizada
- Periapical / Interproximal / Bite Wing
- Levantamento Periapical
- Carpal
- ATM
- Telerradiografia
- Moldagem
- Fotos Intraorais/Extraorais
- Escaneamento Intraoral
- Modelo Ortodôntico

## Exames NÃO Aceitos

Nenhum - todos os exames são aceitos.

## Tipo de Guia

**AMBAS** - Aceita guia física em papel OU guia digital (foto/lançada no sistema).

## Validade da Guia (Pass Card)

**15 dias** a partir da data de emissão do Pass Card.

## PASS CARD - AUTORIZAÇÃO OBRIGATÓRIA

### O que é o Pass Card?

É uma **autorização obrigatória** que o Sempre Odonto exige para realizar **qualquer exame**.

### Quando cliente menciona SEMPRE ODONTO ou CAEDU:

**SEMPRE perguntar**: "Você já tem o Pass Card (autorização)?"

### Se NÃO tem Pass Card:

**RESPOSTA LITERAL:**

"Para fazer exames pelo Sempre Odonto, é necessário ter o Pass Card (autorização prévia). O contato para solicitar fica atrás da carteirinha, você pode ligar lá ou pedir para o seu dentista credenciado solicitar. Assim que tiver a autorização, você pode vir fazer o exame aqui. Posso te ajudar com mais alguma coisa?"

### Se JÁ tem Pass Card:

Seguir normalmente com informações sobre o exame.

### Como conseguir o Pass Card:

- Contato fica **atrás da carteirinha** (física ou digital)
- **Dentista credenciado** pode solicitar e emitir, OU
- **Próprio beneficiário** pode ligar e solicitar

### Resumo Pass Card:

| Item | Informação |
|------|------------|
| Validade | 15 dias |
| Quem pode emitir | Dentista credenciado OU Próprio beneficiário |
| Obrigatoriedade | Obrigatório para TODOS os exames |
